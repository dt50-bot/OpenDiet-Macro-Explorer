# OpenDiet-Macro-Explorer

A transparent Python analysis tool designed to explore macronutrient distributions across various diets (Vegan, Keto, Paleo, DASH, Mediterranean) and global cuisines. 

## Problem Solved
Commercial diet tools often act as proprietary "black boxes" that predominantly recommend Western-centric meals. This repository provides a transparent, fair, and culturally inclusive way to filter and discover recipes based on strict macronutrient needs without locking users out of their preferred cultural cuisines. 

## Ethical Compliance (ACM Code of Ethics)
- **Transparency & No Harm:** We provide raw data filtering, not medical prescriptions. 
- **Fairness:** Highlights 19 distinct global cuisine types equally to prevent cultural dietary bias. 

## Installation & Usage
1. Clone the repository: `git clone https://github.com/yourusername/OpenDiet-Macro-Explorer.git`
2. Install dependencies: `pip install pandas`
3. Run the analyzer: `python src/analyzer.py`

## Disclaimer
*This tool is for educational and analytical purposes only. It is not intended to provide medical or dietary advice. Always consult with a qualified healthcare provider regarding dietary changes.*
